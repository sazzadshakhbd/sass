2. How to setup sass?
   a) npm i -g sass [Globally setup sass command]
   b) sass --version [check sass version]
   c) create sass folder and creats style.scss file (Create sass file and folder)
   d) sass sass/style.scss css/style.css (Compile sass to css)
   e) sass --watch sass/style.scss css/style.css (Compile Continue...)
   f) Project setup done
3. How to setup sass using vscode editor?
   a) Download this extension -- Live Sass Compiler - Ritwick Dey
   b) Click Watch Sass (Below section code editor)
